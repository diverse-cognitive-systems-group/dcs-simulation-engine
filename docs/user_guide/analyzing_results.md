# Analyzing Results

All engine runs produce a `results/` folder that contains database files with all the structured data from the run, logs and metadata.

The a report can be autogenerated using the command below and exported as a pdf or zip file with all raw data used to reproduce the report and run the engine again with the same version and configurations.

```bash
# TODO - update this command when CLI is finalized
dcs report generate
```

For detailed manual exploration of result data, you can checkout the code and use the manual analysis notebooks provided as templates. The loaders read in all the structured data from the database files and logs and make it easily accessible in pandas dataframes for analysis and visualization.
