# Validator Pass/Fail Test Cases

These test cases are *used in live tests to identify what validator models successfully detect and what they miss* under current prompt and model configurations.

IMPORTANT NOTE: Different prompt–model combinations will yield varying results. While one goal of these tests is to **support iterative refinement and optimization of prompts**, an equally important goal is to **surface failure cases clearly for reporting**. As such, not all cases are expected to pass consistently—these tests function primarily as a diagnostic tool.
