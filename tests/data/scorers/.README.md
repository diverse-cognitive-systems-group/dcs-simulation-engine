# SCORER Pass/Fail Test Cases

These test cases are *used in live tests to validate if scorers are scoring as expected* under current prompt and model configurations.

IMPORTANT NOTES: 

- Different prompt–model combinations will yield varying results. While one goal of these tests is to **support iterative refinement and optimization of prompts**, an equally important goal is to **surface failure cases clearly for reporting**. As such, not ALL cases are expected to pass consistently—these tests function primarily as a diagnostic tool.

- Scorers produce a tier (0-3), a score (0-100), and reasoning that justifies the score. Because these tests are automated, **only the tier and score are tested**.
